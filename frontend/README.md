# Visao Coding Challenge
## Frontend

1. Go to frontend directory.
    - cd frontend
   
2. Install dependencies.
   - npm install
   
3. Run the application.
    - npm run dev 
    - running on http://localhost:5173/

## Tests
1. Go to frontend directory.
   - cd frontend

2. Run test command.
   - npm test
